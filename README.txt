Objectives:
1. Collect the key to get in the castle
2. Collect the key to open a chest
3. Lover the bridge with the gear from the chest

Controls:
WASD - move
Space - jump
Shift - hold while moving to run

Right Mouse Button - hold to aim
Left Mouse Button - press while aiming to shoot
F - backstab the enemy
Left Mouse Button - press while not aiming for sword attack

R - restart the game
Esc - exit the game
